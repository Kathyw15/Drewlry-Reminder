# 🎁 AI Gifting Reminder

一个美观的AI Gifting提醒应用，帮助您管理朋友和家人的重要日子。

## 功能特性

- ✨ **添加提醒** - 记录朋友/家人的生日、纪念日等重要日子
- 📅 **智能提醒** - 自动计算距离重要日子的天数
- 🎯 **分类筛选** - 按事件类型（生日、纪念日、节日等）筛选
- 🔔 **即将到来** - 自动显示30天内的重要日子
- ✏️ **编辑管理** - 轻松编辑和删除提醒
- 💾 **本地存储** - 数据保存在浏览器本地，无需服务器

## 技术栈

- React 18
- TypeScript
- Vite
- date-fns (日期处理)
- CSS3 (现代化UI设计)

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

应用将在 `http://localhost:5173` 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 使用说明

1. **添加提醒**：点击"添加提醒"按钮，填写姓名、关系、事件类型和日期
2. **查看提醒**：所有提醒按距离日期排序显示
3. **筛选提醒**：使用顶部标签筛选不同类型的事件
4. **编辑提醒**：点击卡片上的"编辑"按钮修改信息
5. **删除提醒**：点击"删除"按钮移除不需要的提醒
